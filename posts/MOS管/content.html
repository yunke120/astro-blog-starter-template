MOS 管 — 费曼学习法笔记

学习日期：2026-06-28 | 方法：费曼学习法

一、核心问题

理解 MOS 管（金属-氧化物-半导体场效应晶体管）的工作原理、分类及其在电路中的作用。

二、关键概念

1. MOS 管是什么

MOS 管（MOSFET）是一种电压控制的电子开关，有三个引脚，通过 Gate 和 Source 之间的电压差（Vgs）控制 Drain 和 Source 之间是否导通。

2. 三个引脚

引脚 符号 作用
Source (S) 源极 电流出口（NMOS 通常接地，PMOS 通常接电源）
Drain (D) 漏极 电流入口
Gate (G) 栅极 控制极——用电压控制是否导通

3. 两种类型

类型 导通条件 Source 接法
NMOS Vgs > Vth（Vth 通常 > 0，约 0.5~0.8V） 通常接 GND（地）
PMOS Vgs < Vth（Vth 通常 < 0） 通常接 VDD（电源正极）
⚠ 注意： Vgs > 0 不等于导通！必须超过阈值电压 Vth 才导通。

4. 两种工作状态

状态 条件 特性
可变电阻区（线性区） Vgs 刚超过 Vth 像可调电阻，Vgs 越大电阻越小
饱和区（开关区） Vgs 远大于 Vth 像开关闭合，全导通

三、追问回顾

追问 1：Vgs 指什么？

答案： Vgs = Vgate − Vsource。不是任意两点之间的电压。

追问 2：PMOS 为什么 Source 接 VDD？

答案： PMOS 控制正电源通断，NMOS 控制接地回路通断。两者搭配组成 CMOS（互补 MOS），一个 PMOS 拉高电压、一个 NMOS 拉低电压，效率高、功耗低。

追问 3：MOS 管只有通/断两种状态吗？

答案：不是。 Vgs 刚超过 Vth 时电流可调（线性区），Vgs 足够大时才完全导通（饱和区）。这也是 MOS 管能做模拟放大器的原因。

四、提炼总结

一句话版： MOS 管是一个电压控制的三脚开关，用 Gate 和 Source 之间的电压差（Vgs）控制 Drain 到 Source 是否导通。NMOS 的 Vgs 大于阈值时导通（Source 通常接地），PMOS 的 Vgs 小于阈值时导通（Source 通常接电源）。当 Vgs 刚刚超过阈值时，电流可调；当 Vgs 远大于阈值时，像普通开关一样全通。

最佳类比：感应水龙头 🚰

  • Drain（漏极） → 水流入口
  • Source（源极） → 水龙头出口
  • Gate（栅极） → 感应器（感应到你的手 → 开阀）
  • NMOS → 感应器感应到信号（手靠近）→ 放水
  • PMOS → 感应器感应到信号消失（手远离）→ 放水（和 NMOS "反着来"）
  • 阈值 Vth → 手离感应器必须足够近才触发（刚触发时水流小，贴上去才全开）
